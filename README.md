# py-om

**Doom Clone made with Ursina Engine (Python 3)**

***This game is still in Development.***

## HOW TO PLAY

### GNU/Linux & Mac
`pip3 install -r requirements.txt`

`python3 game.py` (in the py-om folder)

### Windows

*idk*

'W, A, S, D' to move

'LEFT MOUSE' to shoot

'SPACE' to jump

'SHIFHT' to run
