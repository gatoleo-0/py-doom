# py-doom
**Doom Clone made with Ursina Engine (Python 3.8)**

***This game is still in Development.***

## HOW TO PLAY

### GNU/Linux
`pip3 install -r requirements.txt`

`python3 game.py` (in the py-doom folder)

### Windows

*idk*

##### Buttons
'x' crouch

'w, a, s, d' to move

'left mouse' to shoot

'space' to jump
